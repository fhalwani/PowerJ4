Workload

This screen shows the various metrics that measure pathologists workload, including the total number of cases, blocks and slides, as well as the fulltime equivalence units (FTE) for CAP-ACP, W2Q, RCP, CPT and Code5 workload measurement systems (WMS). You can define what cases to be included in this report using the facility, accession and specialty templates in the setup menu.

The report contains 6 pie charts that show the breakdown by cases and the 5 WMS from any row selected from the tree table row that you select.

Dates:

The report can generate daily, weekly, monthly, quarterly and annual reports depending on the date that the cases were finalized. the time is from midnight of the start date to midnight of the end range; therefore to collect data of an entire week, select a date from Monday to Monday. Similarly, for an annual report, set the date from Jan 1st to Jan 1st.

Rows:

The report hierarchy can be specified by the drop down menu at the top. The default is facility, specialty, subspecialty, procedure, pathologist. To change the hierarchy to specialty then facility only, begin by setting pathologist to 0, then procedure to 0, then subspecialty to 0, then specialty to 1 (facility will move automatically to 2).

Data Table:

The table shows totals on the top (root), followed by the breakdown by the hierarchy that is defined in the rows (branches). The tree can be expanded or collapsed using the mouse button or the keyboard:

Right ArrowExpand the selected branch
Left ArrowCollapse the selected branch
Plus SignExpand the selected branch
Minus SignCollapse the selected branch
Ctrl-RightExpand all branches
Ctrl-LeftCollapse all branches
Ctrl-PlusExpand all branches
Ctrl-MinusCollapse all branches
Alt-RightExpand all branches under the selected branch
Alt-LeftCollapse all branches under the selected branch
Alt-PlusExpand all branches under the selected branch
Alt-MinusCollapse all branches under the selected branch
