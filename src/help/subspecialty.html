PowerJ Setup - Subspecialties

Used to group cases and specimens for subspecialty practices, like breast, GI and Gyne. Subspecialties are grouped by the specialties table. The subspecialty table has the following fields:

  • Name: up to 8 characters.

  • Description: up to 32 characters

  • Specialty: select one from the parent specialty table by clicking the embeded drop-down menu.

You can add a new subspecialty by first scrolling the list on the left side of the screen to the bottom with the arrows or mouse. You will see a blank item that you can select, then fill-in the required fields.
