Specimen Editor

This screen allows you to view or edit the specimen(s) templates of a specific case in Powerpath without any side effects. No intrinsic triggers such as initiating or deleting any specific specimen template orders will be activated. To proceed, enter the case number in the field provided, then press the key. If such accession number exists in Powerpath, then the specimens table and events table will be populated from the corresponding data in Powerpath

To modify a specimen, press the drop-down menu in the code column, and scroll up or down to the correct specimen template. Press the key again to save the changes.

To undo the changes, you must re-select the original code if you have already pressed the key, or press the key if you have not pressed the key yet.
