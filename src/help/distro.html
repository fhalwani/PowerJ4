Distribution

This screen shows the distribution of the workload among pathologists broken down into the various subspecialties. You can view the data by cases, specimens, blocks, slides, as well as the fulltime equivalence units (FTE) for CAP-ACP, W2Q, RCP, CPT and Code5 workload measurement systems (WMS).

Filter toolbar:

The second toolbar on top of the screen below the quick access toolbar and above the charts, allows you to filter the data by facility, specialty. It also allows you to set a date range for the report. Finally, you can select the data it displays from the data drop-down menu. After you make all the appropriate selections, press the Ok button to refresh the report.

Dates:

The report can generate daily, weekly, monthly, quarterly and annual reports depending on the date that the cases were finalized. the time is from midnight of the start date to midnight of the end range; therefore to collect data of an entire week, select a date from Monday to Monday. Similarly, for an annual report, set the date from Jan 1st to Jan 1st.

Data:

The report data can be specified by the drop down menu at the top. The default is by cases, which can be changed to specimens, blocks, slides or any of the WMS.
