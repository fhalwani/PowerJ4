PowerJ Setup - Orders Groups

This table breaks down the pathology orders (blocks, H&E slides, special stains, IHC, EM, etc) into a smaller set of groups that differ in their workload values. So considering that the average pathology system has more than 1,000 different stains, this table groups them into 20-30 items. This makes for more efficient maintenance and memory management. For example, all the IHC stains are divided into 4 groups:


  • IHCQNT: stains that require quantitative interpretation (ER, PR, Ki-67).

  • IHCMLT: multi-color IHC stains that use 2 or more antibodies with different colors that require independant interpretation (coded differently by CPT). Note that Pan-cytokeratin uses multiple antibodies but 1 color only and therefore does not qualify into this category

  • IHCCD: Cluster differentiation antibodies used in hematopoeitic and lymphoproliferative disorders (coded differently by W2Q).

  • IHC: the rest of IHC stains that do not belong to the above 3 categories.

Note: the name of of coder system can be modified here.


You can review and edit the parameters of each group here whenever a workload system revision is available. You do so by editing the following fields:


  • Name: up to 8 characters.

  • Description: up to 64 characters

  • Type: select one of the 12 more basic order types (block, H&E, SS, IHC).

  • Code 1 (CAP): Select how to process the workload units of this order from its respective coder table.

  • Code 2 (W2Q): Select how to process the workload units of this order from its respective coder table.

  • Code 3 (RCP): Select how to process the workload units of this order from its respective coder table.

  • Code 4 (CPT): Select how to process the workload units of this order from its respective coder table.
