PowerJ Setup - Specialties

Used to group specimens and subspecialties into autopsy, cytology or surgical pathology.

Each entry has these fields that you use to control if cases from this facility are managed by PowerJ data mining analysis:

  • Workflow: includes cases from this specialty in workflow, backlog, routing and histology reports.

  • Workload: includes cases from this specialty in workload, turn-around time, forecast and distribution reports.

  • Code Specimen: should be disabled whenever the specimen itself has no value and only the orders are to be coded in the workload (consults or molecular reports).

Facility, Specialty and Accession tables work together in defining the cases to be included or exluded in the workflow or workload tracking. You can deactivate all the cases from a facility in the Facility table. Vice versa, you can deactivate all the autopsy cases in this table. But to activate/deactivate certain subspecialties (consults, bone marrow) of a certain facility, you must also select/deselect it in the Accession table.

You can add a new specialty by first scrolling the list on the left side of the screen to the bottom with the arrows or mouse. You will see a blank item that you can select, then fill-in the required fields.
