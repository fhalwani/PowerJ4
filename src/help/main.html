Main Screen

PowerJ main screen includes the following components:

  • The menu bar to open the different panels with shortcuts listed next to each item. To activate the menu use the mouse or press the alt key.

  • The toolbar to access the most common reports. If you hover the mouse over its buttons, you will see a popup tooltip describing the attached panel

  • The status bar at the bottom that displays important system messages and a progress bar when the system is running a long time-consuming task like updating data.


It is recommended not to close the application while the progress bar is active, as it may cause data loss or corruption. This is especially critical for the desktop version of the software.
