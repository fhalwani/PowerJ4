PowerJ Setup - Coders

There are 5 available workload coders in PowerJ:


  • The Canadian Association of Pathologists (CAP)

  • The Ontario Work2Quality (W2Q)

  • The British Royal College of Pathologists (RCP)

  • The American Medical Association Current Procedural Terminology (CPT)

  • Code5: A specimen-based average of the 4 above coders calculated and managed by PowerJ.


You can review and edit the parameters of each system here whenever an updated version is available. You do so by editing the following fields:


  • Name: up to 16 characters.

  • Description: up to 256 characters

  • Rule: select one of the 57 rules applicable to the code from the drop-down menu. See a detailed description of the codes here with some examples.

  • Count: an integer applicable to the rule selected.

  • Value 1: a double precision number applicable to the rule selected.

  • Value 2: a double precision number applicable to the rule selected.

  • Value 3: a double precision number applicable to the rule selected.

Note: The text field at the bottom of the screen displays a description of the rule that you have selected.


Additional Examples:

  • Autopsy Complete (CAP): Rule 'SP_CA_INCL', Count '1', Value1 '32.4', Value2 '0', value3 '0'. Codes the entire case, no specimens and no orders

  • Medical Biopsy W865 (W2Q): Rule 'SP_SP_FIXED', Count '1', Value1 '2.12', Value2 '0', value3 '0'. Codes the specimen plus additional orders

  • Skin Biopsy R1a (CAP): Rule 'SP_FR_EV_X', Count '5', Value1 '0.95', Value2 '0', value3 '0'. Codes the specimen by the number of fragments plus additional orders.

  • Prostate Biopsy R2b (CAP): Rule 'SP_FR_12X', Count '1', Value1 '0.95', Value2 '0.238', value3 '9'.

  • Malignant Resections R5 (CAP): Rule 'SP_CA_BL_12X', Count '8', Value1 '1.9', Value2 '0', value3 '0.238'.

  • Radical Resections R9 (CAP): Rule 'SP_CA_BL_EV_X', Count '1', Value1 '0.238', Value2 '9', value3 '0'.

  • Additional H&E Levels R6A (CAP): Rule 'OR_AFT_EV_X', Count '3', Value1 '0.119', Value2 '0', value3 '0'.

  • Frozen Sections (CAP): Rule 'SP_SP_123', Count '2', Value1 '2.85', Value2 '1.9', value3 '1.9'.


You can add a new code by first scrolling the list on the left side of the screen to the bottom with the arrows or mouse. You will see a blank item that you select, then fill-in the required fields.


You can rename, de-activate, re-activate any of these systems, and set their FTE equivalence in the PowerJ setup screen.


You can also replace an entire workload management system by another one of your choice, however this is not a simple task and requires careful planning and evaluation of its compatibility with PowerJ.
