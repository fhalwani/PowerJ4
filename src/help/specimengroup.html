PowerJ Setup - Specimens Groups

This table defines all possible specimens encountered in Pathology and how they will be treated in calculating the workload. They are compiled from the workload management systems that PowerJ supports (CAP, W2Q, CPT, RCP) with instructions on how to code each specimen by these workload systems. It is also used by PowerJ to classify the cases by subspecialty and procedure. Your institution-specific specimens reside in another table called Specimens Master that you can use to link your specimens to the ones here. This makes for more efficient maintenance and memory management.


Each entry has the following fields that you use to control how the workload is measured:

  • Description: the name given to the specimen.

  • Procedure: a link to the Procedures table (biopsy, excision, etc).

  • Subspecialty: a link to the Subspecialty table (GI, GU, GYN).

  • Has LN: does the specimen have lymph nodes embeded within it (colectomy, gastrectomy). It is required by CAP system to promote a malignant specimen from R5 to R9.

  • Code5: The workload value of the specimen based on the average of the 4 systems.

  • Table: Defines what workload codes to use for this specimen if benign, malignant or radical. For example, The CAP rule for a breast simple mastectomy is defined as R4 if benign, but R5 if malignant..
