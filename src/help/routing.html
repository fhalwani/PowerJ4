Routing

This report displays the cases routed to each pathologist on each day over the past 5 days.


Dates:

Use the list on the left side of the screen to select which date you wish to view. If the time now has passed the cutoff time that is displayed with the date and can be modified in the PowerJ Setup screen, then tomorrow's date will also be shown at the top. The last (earliest) date in the list is not selectable, and if you do, the tables will be blank.

Cases Table:

The bottom table shows the cases that have been routed. The table is initially sorted by the accession date, but you can change the sort order by clicking on the column header of your choice.


Summary Table:

The table in the middle of the screen shows a summary of case routing by pathologist, along with their initials, full name, number of cases, number of specimens, number of slides, number of workload units (in minutes), and the FTE equivalence. The table is initially sorted by wrokload in ascending order, but you can change the sort order by clicking on the column header of your choice.


Workload Chart:

The chart displays the total number of FTE units assigned to each pathologist, and is sorted in ascending order.


Filter toolbar:

The second toolbar on top of the screen below the quick access toolbar and above the charts, allows you to filter the data by facility, specialty or subspecialty.
