Forecast

This screen shows the annual volume of each specimen from the years available in the system. For each year, the total count, blocks, slides, as well as the fulltime equivalence units (FTE) for CAP-ACP, W2Q, RCP, CPT and Code5 workload measurement systems (WMS).

The report contains 6 pie charts that show the breakdown by specimens and the 5 WMS from any row selected from the tree row that you select. It also extrapolates the data into the next 3 years (green squares).

Rows:

The report hierarchy can be specified by the drop down menu at the top. The default is facility, specialty, subspecialty, procedure, specimen. To change the hierarchy to specimen then facility only, begin by setting procedure to 0, then subspecialty to 0, then specialty to 0, then specimen to 1 (facility will move automatically to 2).

Data Tree:

The tree shows the total on the top (root), followed by a breakdown by the hierarchy that is defined in the rows (branches). The tree can be expanded or collapsed using the mouse button or the keyboard:

Right ArrowExpand the selected branch
Left ArrowCollapse the selected branch
Plus SignExpand the selected branch
Minus SignCollapse the selected branch
Ctrl-RightExpand all branches
Ctrl-LeftCollapse all branches
Ctrl-PlusExpand all branches
Ctrl-MinusCollapse all branches
Alt-RightExpand all branches under the selected branch
Alt-LeftCollapse all branches under the selected branch
Alt-PlusExpand all branches under the selected branch
Alt-MinusCollapse all branches under the selected branch
