Finals

This screen allows you to review how PowerJ is analyzing and coding the workload of cases that have been reported and make sure it is done correctly. The information is displayed in 5 tables and 1 notepad:

Cases:

A list of the most recent cases, up to 10,000 at a time. Use the filter toolbar to select which cases to display. The table has the following fields:

  1. NO: The case number.

  2. FAC: Facility.

  3. SPY: Specialty.

  4. SUB: Subspecialty.

  5. PROC: Procedure.

  6. SPEC: The label of the main specimen.

  7. CAP: The workload units for the entire case from the Canadian Association of Pathologists workload system.

  8. W2Q: The workload units for the entire case from the Work2Quality workload system (Ontario).

  9. RCP: The workload units for the entire case from the Royal College of Pathologists workload system (UK).

  10. CPT: The workload units for the entire case from the Current Procedural Terminology workload system (USA).

  11. Code5: The average of the above 4 workload systems.

  12. SPECS: number of specimens.

  13. BLKS: Number of blocks for the entire case.

  14. SLDS: Number of slides for the entire case.

  15. H&E: Number of hematoxylin and eosin slides for the entire case.

  16. SS: Number of special stains for the entire case.

  17. IHC: Number of immunohistochemistry slides for the entire case.

  18. MOL: Number of molecular tests for the entire case.

  19. SYNP: Number of synoptic reports for the entire case.

  20. FS: Number of frozen section procedures for the entire case.

  21. ACCESS: The date and time the case was accessioned.

  22. GROSS: The date and time the case was grossed.

  23. EMBED: The date and time the blocks were embedded.

  24. MICRO: The date and time the blocks were microtomed.

  25. ROUTE: The date and time the case was routed to the pathologist.

  26. FINAL: The date and time the report was issued.

  27. GRNM: The initials of the person who grossed the case.

  28. EMNM: The initials of the technician who embedded the blocks.

  29. MINM: The initials of the technician who microtomed the blocks.

  30. RONM: The initials of the technician who routed the slides.

  31. FINM: The initials of the pathologist who reported the case.

  32. GRTA: The turnaround time of grossing the case.

  33. EMTA: The turnaround time of embedding the blocks.

  34. MITA: The turnaround time of microtoming the blocks.

  35. ROTA: The turnaround time of routing the slides.

  36. FITA: The turnaround time of reporting the case.

Specimens:

The specimens that belong to the case selected (highlighted) from the cases table above it. This table has the following fields:

  1. SPEC: The label of this specimen.

  2. CAP: The workload units for the specimen from the Canadian Association of Pathologists workload system.

  3. W2Q: The workload units for the specimen from the Work2Quality workload system (Ontario).

  4. RCP: The workload units for the specimen from the Royal College of Pathologists workload system (UK).

  5. CPT: The workload units for the specimen from the Current Procedural Terminology workload system (USA).

  6. Code5: The average of the above 4 workload systems for this specimen.

  7. BLKS: Number of blocks for the specimen.

  8. SLDS: Number of slides for the specimen.

  9. FRAG: Number of tissue fragments for the specimen (used for CAP workload in some specimens).

  10. H&E: Number of hematoxylin and eosin slides for the specimen.

  11. SS: Number of special stains for the specimen.

  12. IHC: Number of immunohistochemistry slides for the specimen.

  13. MOL: Number of molecular tests for the specimen.

Orders:

The orders that belong to the specimen that you selected (highlighted) from the specimens table above it. It has the following fields:

  1. ORDER: The category of the order as defined in this table.

  2. QTY: Quantity.

  3. CAP: The workload units for the order from the Canadian Association of Pathologists workload system.

  4. W2Q: The workload units for the order from the Work2Quality workload system (Ontario).

  5. RCP: The workload units for the order from the Royal College of Pathologists workload system (UK).

  6. CPT: The workload units for the order from the Current Procedural Terminology workload system (USA).

Frozen sections:

The frozen sections that were performed on the specimen selected (highlighted) in the table above it. It has the following fields:

  1. FSEC: The pathologist name who performed the procedure.

  2. CAP: The workload units for the frozen from the Canadian Association of Pathologists workload system.

  3. W2Q: The workload units for the frozen from the Work2Quality workload system (Ontario).

  4. RCP: The workload units for the frozen from the Royal College of Pathologists workload system (UK).

  5. CPT: The workload units for the frozen from the Current Procedural Terminology workload system (USA).

  6. Code5: The average of the above 4 workload systems for this frozen.

  7. BLKS: Number of blocks for the frozen.

  8. SLDS: Number of slides for the frozen.

Additionals:

Additional clinical procedures that were performed by a different pathologist during of after finalizing the report. Examples are addenda, amendments, second opinions, reviews, correlations and presentation at tumor board. The workload units of these procedures are assigned to the pathologist who performed the procedure, wether done by the same or a different pathologist than the one who reported the case. This table has the following fields:

  1. ADDL: The name of the clinical procedures.

  2. CAP: The workload units for the procedure from the Canadian Association of Pathologists workload system.

  3. W2Q: The workload units for the procedure from the Work2Quality workload system (Ontario).

  4. RCP: The workload units for the procedure from the Royal College of Pathologists workload system (UK).

  5. CPT: The workload units for the procedure from the Current Procedural Terminology workload system (USA).

  6. Code5: The average of the above 4 workload systems for this procedure.

  7. STAFF: The initials of the pathologist who performed the procedure.

  8. DATE: the date and time of the procedure.

Notepad:

This is a text describing the steps that PowerJ used to code the workload of the specimen. This text is deleted automatically after one year of finalizing the case.

Filter toolbar:

The second toolbar on top of the screen below the quick access toolbar and above the charts, allows you to filter the data by facility, specialty or subspecialty.
